# PlegaOS25-Plus
PlegaOS25 Plus - Welcome to PlegaOS25 Plus! This is a guide on how to use special apps, other then media center and clock and all the other normal apps, PlegaOS25 Comes with some extras, the browser, and shop. The shop allows you to download over 20 apps online and offline. And the browser lets you search the yep.com search engine.
